HTML Notes

Basic Structure

Every HTML file should start with a <!DOCTYPE html> declaration and contain an <html> element. The <head> element provides essential information for the page, including the <title>.

HTML Elements, Tags, and Indentations

HTML elements are building blocks of web pages. Tags, enclosed in angle brackets, define elements. Proper indentation enhances readability.

Text Formatting

  • <b>: Bold text
  • <i>: Italic text
  • <u>: Underlined text
  • <s>: Strikethrough text

Paragraphs and Line Breaks

Use <p> for paragraphs and <br> for line breaks.

Lists

  • Unordered lists use <ul> and <li>.
  • Ordered lists use <ol> and <li>.

Links and Images

Use the <a> tag to create hyperlinks and the <img> tag to include images. Attributes like href and src are used for links and images respectively.

Styles and Attributes

Attributes like class and id help organize elements, while style allows inline CSS for presentation.

Comments

HTML comments (<!-- -->) are useful for documentation or temporarily removing code.

Forms and User Input

The <form> and <input> elements allow user interaction. Inputs support various types like text, number, checkbox, radio, and more. Validation attributes ensure correct data submission.

HTML5 Semantic Elements

HTML5 introduced semantic elements to enhance the meaning of web pages. These include:

  • <header>: Defines introductory content or navigation links.
  • <nav>: Specifies navigation links.
  • <main>: Represents the main content of the page.
  • <section>: Groups related content.
  • <article>: Contains self-contained content.
  • <figure> and <figcaption>: Represent images or diagrams and their captions.
  • <footer>: Defines footer content.

Semantic Structure Example

<body> <header> <nav></nav> </header> <main> <section> <article> <figure> <figcaption></figcaption> </figure> </article> </section> </main> <footer></footer> </body>
